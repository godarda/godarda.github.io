Strings Concatenation
>>> s1="Koding"
>>> s2="window"
>>> s1+s1
'KodingKoding'

>>> s1+s2
'GoDarda'

>>> s1*5
'KodingKodingKodingKodingKoding'

>>> s1,s2
('Koding', 'window')
Escape Characters
>>> gd="Koding\nwindow"
>>> gd
'Koding\nwindow'

>>> gd="Koding\nwindow"
>>> print(gd)
Koding
window
 
>>> gd=r"Koding\nwindow"
>>> print(gd)
Koding\nwindow
 
>>> print('\"GoDarda\"')
"GoDarda"

>>> gd="Koding\twindow"
>>> print(gd)
Koding	window
