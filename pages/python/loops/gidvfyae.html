Python program to print the right half pyramid pattern using alphabets
{{ site.pyfile }}
{%- highlight python -%}
anum=65
for n in range(1,7):
    for i in range(n):
        print(chr(anum),end="")
        anum+=1
    print()
{%- endhighlight -%}
{{ site.output }}

BC
DEF
GHIJ

PQRSTU
