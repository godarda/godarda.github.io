touch and cat commands to create the files
godarda@gd:~$ ls
a.out  GD.class  GD.java  Desktop    Downloads	Pictures  Templates  Weekdays
gd.c   gd.csv	 gd.py	  Documents  Music	Public	  Videos

godarda@gd:~$ touch Ubuntu.txt Fedora.txt
godarda@gd:~$ ls
a.out	  gd.csv   Desktop    Fedora.txt  Public      Videos
gd.c	  GD.java  Documents  Music	  Templates   Weekdays
GD.class  gd.py    Downloads  Pictures	  Ubuntu.txt

godarda@gd:~$ cat > KaliLinux
^C
godarda@gd:~$ ls
a.out	  gd.csv   Desktop    Fedora.txt  Pictures   Ubuntu.txt
gd.c	  GD.java  Documents  KaliLinux   Public     Videos
GD.class  gd.py    Downloads  Music	  Templates  Weekdays

godarda@gd:~$ file Fedora.txt gd.c gd.csv
Fedora.txt: empty
gd.c:       C source, ASCII text
gd.csv:     CSV text
rm command to remove the file(s)
godarda@gd:~$ rm KaliLinux Weekdays
godarda@gd:~$ ls
a.out	  gd.csv   Desktop    Fedora.txt  Public      Videos
gd.c	  GD.java  Documents  Music	  Templates
GD.class  gd.py    Downloads  Pictures	  Ubuntu.txt
rm command with a regular expression to remove multiple files of the same extension
godarda@gd:~$ rm *.txt
godarda@gd:~$ ls -1
a.out
gd.c
GD.class
