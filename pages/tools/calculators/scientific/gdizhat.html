The Trigonometry Calculator computes the values of standard trigonometric functions for a given angle. It features two input modes: Standard Angles for common values and Custom Input for specific calculations.

How to Use
  • Input Mode: Toggle "Standard Angles" to switch between selecting common angles and entering custom values.
  • Standard Mode: Select a predefined angle (e.g., 30, 45, 90) and unit. Enable "Show Fractions" to see exact values (e.g., √3/2).
  • Custom Mode: Enter any numerical angle and select the unit (Degrees, Radians, or Gradians).
  • View Results: Sine and Cosine values are shown by default. Use the "Select Tiles" dropdown to display Tangent, Secant, Cosecant, and Cotangent.
