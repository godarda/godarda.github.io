The Fuel Consumption Converter is a practical tool designed to compare fuel efficiency across different international standards. Whether you are calculating trip costs, comparing vehicle specifications, or converting between metric and imperial systems, it provides accurate results instantly.

How to Use
  • Input Value: Enter the fuel consumption value you wish to convert in the input field.
  • Select Unit: Choose the starting unit (e.g., MPG (US), L/100km) from the dropdown next to the input.
  • Customize Results: Use the "Select Tiles" dropdown to choose which target units to display.
Supported Units

This tool supports conversion between the following fuel efficiency units:

  • Metric: Liters per 100 Kilometers (L/100km), Kilometers per Liter (km/L)
  • Imperial / US: Miles per Gallon (US), Miles per Gallon (Imperial)
