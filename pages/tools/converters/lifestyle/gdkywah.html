The Cooking Converter is a handy tool designed to simplify kitchen conversions. Whether you're following an international recipe or scaling up a meal, it provides instant and accurate conversions between common cooking measurements.

How to Use
  • Input Value: Enter the quantity you wish to convert in the input field.
  • Select Unit: Choose the starting unit (e.g., Cup, Tablespoon) from the dropdown next to the input.
  • Customize Results: Use the "Select Tiles" dropdown to choose which target units to display.
Supported Units

This tool supports conversion between the following cooking units:

  • Metric: Milliliter (ml), Liter (l)
  • US Customary: Teaspoon (tsp), Tablespoon (tbsp), Fluid Ounce (fl oz), Cup, Pint (pt), Quart (qt), Gallon (gal)
  • Imperial: Imperial Tablespoon, Imperial Cup, Imperial Fluid Ounce
