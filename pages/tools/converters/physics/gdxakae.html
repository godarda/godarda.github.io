The Illuminance Converter is a practical tool designed to perform instant conversions between various units of illuminance (luminous flux incident on a surface). Whether for lighting design, photography, or architectural engineering, it provides accurate results across a wide range of light intensity measurements.

How to Use
  • Input Value: Enter the illuminance value you wish to convert in the input field.
  • Select Unit: Choose the starting unit (e.g., Lux, Foot-candle) from the dropdown next to the input.
  • Customize Results: Use the "Select Conversion" dropdown to choose which target units to display.
Supported Units

This tool supports conversion between the following illuminance units:

  • SI / Metric: Lux (lx), Kilolux (klx), Millilux (mlx), Nox (nx)
  • CGS: Phot (ph)
  • Imperial / US: Foot-candle (fc)
