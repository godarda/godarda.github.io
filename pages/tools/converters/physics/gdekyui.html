The Torque Converter is a specialized tool designed to perform instant conversions between various units of torque (rotational force). Whether for automotive engineering, mechanical design, or physics calculations, it provides accurate results across a wide range of measurements.

How to Use
  • Input Value: Enter the torque value you wish to convert in the input field.
  • Select Unit: Choose the starting unit (e.g., Newton-meter, Pound-force foot) from the dropdown next to the input.
  • Customize Results: Use the "Select Conversion" dropdown to choose which target units to display.
Supported Units

This tool supports conversion between the following torque units:

  • Metric / SI: Newton-meter (N·m), Dyne-centimeter (dyn·cm), Kilogram-force meter (kgf·m), Kilogram-force centimeter (kgf·cm)
  • Imperial / US: Pound-force foot (lbf·ft), Pound-force inch (lbf·in), Ounce-force inch (ozf·in)
