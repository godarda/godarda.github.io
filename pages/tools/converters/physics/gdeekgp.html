The Radiation Converter is a specialized tool designed to perform instant conversions between units of absorbed radiation dose and equivalent dose. It is essential for health physics, radiology, and nuclear safety calculations.

How to Use
  • Input Value: Enter the radiation value you wish to convert in the input field.
  • Select Unit: Choose the starting unit (e.g., Gray, Sievert) from the dropdown next to the input.
  • Customize Results: Use the "Select Tiles" dropdown to choose which target units to display.
Supported Units

This tool supports conversion between the following radiation units:

  • Absorbed Dose: Gray (Gy), Milligray (mGy), Microgray (µGy), Rad (rad), Millirad (mrad)
  • Equivalent Dose: Sievert (Sv), Millisievert (mSv), Microsievert (µSv), Rem (rem), Millirem (mrem)

Note: Conversions between absorbed dose and equivalent dose assume a Quality Factor (Q) of 1.
