The Frequency Converter is a precise tool designed to facilitate instant conversions between various frequency units. Whether for electronics, acoustics, or mechanical engineering, it provides accurate results across a wide range of cyclic measurements.

How to Use
  • Input Value: Enter the frequency you wish to convert in the input field.
  • Select Unit: Choose the starting unit (e.g., Hertz, Kilohertz) from the dropdown next to the input.
  • Customize Results: Use the "Select Conversion" dropdown to choose which target units to display.
Supported Units

This tool supports conversion between the following frequency units:

  • SI / Metric: Hertz (Hz), Kilohertz (kHz), Megahertz (MHz), Gigahertz (GHz), Terahertz (THz)
  • Rotational / Angular: Revolutions per minute (rpm), Radian per second (rad/s), Degree per second (deg/s)
