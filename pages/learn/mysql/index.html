MySQL Introduction

MySQL is an open-source database management system. A database is a structured collection of data. Since MySQL stores data in tables, it is classified as a Relational Database Management System (RDBMS). It runs on various platforms, including Linux, Windows, and macOS.

To follow this tutorial sequentially, start with MySQL: CREATE a DATABASE and TABLE, and then follow the recommended links at the bottom of each page. Throughout this tutorial, we use MySQL on the Ubuntu Linux distribution.
Prerequisite: Ensure MySQL is installed and running. If not, refer to How to setup, install, and start MySQL on Ubuntu.
