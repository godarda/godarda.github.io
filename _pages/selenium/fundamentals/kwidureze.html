--- permalink: selenium/fundamentals/kwidureze title: How to launch the browser using Selenium Python ---
Selenium Python to launch the Chrome browser
{{ site.pyfile }}
{%- highlight python -%}
from selenium import webdriver

driver = webdriver.Chrome(executable_path='/home/kodingwindow/drivers/chromedriver')
driver.maximize_window()
driver.get("https://kodingwindow.com/testapp/")
driver.close()
{%- endhighlight -%}
